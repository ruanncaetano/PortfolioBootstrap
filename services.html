Projetos

Projetos desenvolvidos ao longo dos meus anos de aprendizado em desenvolvimento de software, aplicando conhecimentos adquiridos durante o curso Técnico em Informática na ETEC Prof. Milton Gazzetti e aprofundados na graduação em Sistemas de Informação na Unoeste, com foco em práticas modernas e tecnologias emergentes.

Página de Links para E-commerce de Itens Esportivos

Este projeto é uma página da web simples criada com HTML e CSS para fornecer links e informações básicas sobre um e-commerce de itens esportivos. Ele serve como um ponto de partida para direcionar os visitantes para diferentes seções ou categorias de produtos esportivos.


TCC - Book Prime

Este projeto é uma página dedicada ao Trabalho de Conclusão de Curso (TCC) que aborda o uso de um "Caderno Virtual" como solução para superar as dificuldades encontradas durante o ensino à distância (EAD) em 2020. A página é desenvolvida com HTML, CSS e JavaScript para proporcionar uma experiência informativa e interativa.


Exercicio de Condição em C++

Este é um projeto de estudo da linguagem C tendo como principal ponto estudar as as condições utilizando o 'IF' e 'ELSE' que são as estruturas de controle de condição são um dos conceitos fundamentais da programação, permitindo que os programadores tomem decisões com base em condições específicas.

Projeto de Estudo em Java: Classes e Listas

Este projeto tem como objetivo explorar o uso de classes e listas em Java, conceitos fundamentais da programação orientada a objetos. Os participantes aprenderão a criar e manipular classes, instanciar objetos e utilizar listas para armazenar e acessar esses objetos, permitindo um gerenciamento eficiente de coleções de elementos e uma estruturação organizada do código.


Projeto de Estudo em Java: Manipulaçõa de Arquivos

Este projeto tem como foco a manipulação de arquivos em Java, incluindo leitura, escrita, criação e atualização de arquivos de texto. O objetivo é capacitar os programadores a trabalhar com dados armazenados em arquivos de maneira eficiente e segura.
